Make Sure You Eat!

Eat

Eating is important to get brain power for your next code session. Make sure you are getting nourished before you start to code, and you will be able to code with intensity!

Make Sure You Sleep!

Sleep

Sleeping will allow your brain to relax and recooperate for your next code session. Make sure you are sleeping in order to give your brain a break between coding sessions in order not to get exhausted and burn out.

Make Sure You Code!

Code

This one is obvious and straight forward. You can't code well if you don't code. Make sure you are putting in hours and setting time aside to code more and more each day!
